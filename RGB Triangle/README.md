# RGB Triangle

Base Project: https://github.com/Dovyski/opengl-demos/tree/master/src/shader-hello-world

## Windows

Open *RGB Triangle.sln* and run the solution

## Linux

### Install dependencies
```
make install-deps
```

### Run
```
make && ./main
```
